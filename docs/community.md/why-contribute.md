For a normal **college student**, this is a great chance to get a **normal PR (pull request)**, which can be hard to get accepted in big organizations.

Most projects here are **simple and beginner-friendly**. If you contribute while you are in college, **seniors can directly help you** with your contributions.

Working on these projects helps you **learn how open-source works**. This experience is very useful for programs like **GSOC**, because many organizations ask for **previous open-source contributions**.

So, contributing here is an **easy way to start open-source**, learn step by step, and build your profile.
